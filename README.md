P14. Transparent Minds - NLP Project

Setup:
conda env create -f nlp_env.yml
conda activate nlp_env

Usage:
Random Button: Gets a random sentence from a filtered multi_nli dataset
https://huggingface.co/datasets/nyu-mll/multi_nli
Sentences are not always good or make sense but are fine for fast testing.

Enter Button: Enter own sentences into the textbox and process them with the button

Switch the different visualizations by using the tabs.
